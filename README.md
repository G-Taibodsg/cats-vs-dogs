# 🐱🐶 深度学习猫狗分类识别项目

一个基于 PyTorch + PyQt5 的猫狗图片分类桌面应用。
模型已训练完成，开箱即用，无需再下载数据集或重新训练。

📦 安装与运行
1. 克隆仓库

```
git clone https://github.com/G-Taibodsg/cats-vs-dogs
cd cats-vs-dogs
```

2. 创建虚拟环境（推荐）

```
python -m venv .venv
```

3. 安装依赖

```
pip install -r requirements.txt
```

4. 运行桌面应用

```
python gui_app.py
```
